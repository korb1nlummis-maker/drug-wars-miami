========================================
   DRUG WARS - Installation Guide
========================================

WHAT IS THIS?
-------------
Drug Wars is a single-page browser game. No installation required -
just open the HTML file in any modern web browser.


QUICK START (Easiest)
---------------------
1. Extract the ZIP file to any folder on your computer
2. Double-click "index.html"
3. The game opens in your default browser
4. Click START GAME and enjoy!

That's it. No servers, no downloads, no accounts needed.


REQUIREMENTS
------------
- Any modern web browser:
  * Google Chrome (recommended)
  * Mozilla Firefox
  * Microsoft Edge
  * Safari
- Sound/speakers recommended (the game has procedural synthwave music!)
- No internet connection required - works fully offline


TROUBLESHOOTING
---------------

Q: The game won't open / shows a blank page
A: Make sure all 5 game files are in the SAME folder:
   - index.html
   - game-engine.js
   - game-ui.js
   - music-engine.js
   - achievements.js
   Do NOT move index.html without the .js files.

Q: No music is playing
A: Click the speaker/music icon in the top-right corner of the game.
   Browsers require a user interaction before playing audio.
   Make sure your system volume is up.

Q: My save game disappeared
A: Save data is stored in your browser's localStorage.
   - Clearing browser data/cookies will erase saves
   - Each browser has its own separate save
   - Use the in-game SAVE button regularly

Q: The game looks weird / elements overlap
A: Try maximizing your browser window or zooming out (Ctrl + minus).
   The game is designed for screens 1024px wide or larger.

Q: Charts/sparklines aren't showing
A: Charts appear after you travel at least once. They need price
   history data to display.


GAME TIPS FOR NEW PLAYERS
--------------------------
- Buy low, sell high - watch the price charts and event messages
- Start in cheap cities and work your way up
- Hire crew from the Black Market for combat help and carrying capacity
- Watch your Heat level - too high and you'll attract police
- The Investigation meter tracks DEA attention - lay low or hire a Lawyer
- Take over territories for passive income and price discounts
- Set up Front Businesses to launder money and reduce investigation
- Save often! The streets are dangerous.
